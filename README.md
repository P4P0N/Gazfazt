# 🚗 GazFazt
Sistema web prototipo para optimizar el proceso de pago de combustible en estaciones de servicio mediante tecnología digital.

---

## 📌 Descripción del proyecto
**GazFazt** es una aplicación web que simula un sistema de pago rápido para gasolina, donde el usuario puede:

- Recargar saldo en su cuenta  
- Realizar pagos de combustible de forma digital  
- Gestionar su cuenta desde la plataforma  
- Identificarse mediante código QR  

El objetivo es reducir tiempos de espera, eliminar el uso de efectivo y mejorar la experiencia del usuario en estaciones de servicio.

---

## 🛠️ Tecnologías utilizadas
Actualmente el proyecto está desarrollado con:

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  

⚠️ **Nota:** Por el momento no está conectado a una base de datos, todo funciona como simulación.

---

## ▶️ Cómo ejecutar el proyecto

Para correr el proyecto en tu computador:

1. Abre la carpeta del proyecto en Visual Studio Code  
2. Ubica el archivo:
   index.html
3. Haz clic derecho sobre el archivo  
4. Selecciona:
   Open with Live Server
5. Automáticamente se abrirá en tu navegador

## 🧭 Flujo de uso actual

### 🏠 Página principal
Se muestra:

- Menú de navegación  
- Sección ¿Cómo funciona?  
- Sección ¿Quiénes somos?  
- Sección Contáctanos  
- Botón de Iniciar sesión  

---

### 🔐 Inicio de sesión
- Actualmente es una simulación  
- No hay conexión real con base de datos  
- Permite acceder a la interfaz interna  

---

### 👤 Panel de usuario
Una vez dentro:

- En la parte superior derecha aparecerá el usuario  
- Se debe dar clic en "Mi cuenta"  
- Se despliega un menú tipo hamburguesa
